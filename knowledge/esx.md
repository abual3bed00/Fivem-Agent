# ESX Framework Documentation (Extended Mode)

ESX is the most popular framework for legacy FiveM scripts.

## Core Functions
- `ESX.GetPlayerFromId(source)`: Get the user object.
- `ESX.GetPlayers()`: Get all players.
- `ESX.ShowNotification(msg)`: Basic notification.

## Events
- `esx:getSharedObject`: Used to get the ESX object in older versions.
- `esx:playerLoaded`: Triggered on player load.
