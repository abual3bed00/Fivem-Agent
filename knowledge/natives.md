# FiveM Natives Knowledge Base

## Ped Natives
- `GetPlayerPed(player)`: Get the ped entity for a player.
- `SetEntityCoords(entity, x, y, z)`: Teleport entity.
- `GiveWeaponToPed(ped, weaponHash, ammo, isHidden, bForceInHand)`: Give weapon.

## UI Natives
- `SetNuiFocus(hasFocus, hasCursor)`: Toggle NUI focus.
- `SendNUIMessage(data)`: Send data to NUI HTML.
- `RegisterNUICallback(name, callback)`: Handle NUI fetch requests.
