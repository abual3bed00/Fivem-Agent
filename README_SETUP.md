# 🎮 FiveM Script Agent - AI Setup

هذا المجلد يحتوي على كود المصدر لتطبيقك.

## 🚀 الخطوات النهائية للتشغيل

1. **تثبيت Node.js**: إذا لم تثبته بعد، حمله من [nodejs.org](https://nodejs.org).
2. **تثبيت المكتبات**: افتح PowerShell في هذا المجلد ونفذ:
   ```powershell
   npm install
   ```
3. **تشغيل التطبيق**: بعد تثبيت المكتبات، نفذ:
   ```powershell
   npm start
   ```

## 📂 هيكل المجلدات
- `renderer/`: واجهة المستخدم (HTML/CSS/JS).
- `agent/`: منطق الذكاء الاصطناعي و RAG.
- `knowledge/`: قاعدة المعرفة التي يتعلم منها الأجينت.
- `templates/`: قوالب السكريبتات الجاهزة.
- `models/`: سيتم تحميل نموذج الذكاء الاصطناعي هنا تلقائياً عند أول تشغيل.
